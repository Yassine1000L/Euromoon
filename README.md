Dit project is een makkelijke en simpele  command line Java applicatie voor de treinmaatschappij Euromoon.  
Het project is gemaakt voor het vak Programming Advanced en dient als proof of concept.
Met de applicatie kan je passagiers registreren, reizen aanmaken, tickets verkopen en boardinglijsten genereren.


-	Wat kan de applicatie?
De applicatie kan de  volgende soorten dingens doen:

- Passagiers registreren  
- Een reis aanmaken tussen 2 stations  
- Een trein koppelen aan een reis  
- Tickets verkopen in 1ste of 2de klasse  
- Controleren of er nog plaatsen beschikbaar zijn  
- Boardinglijsten genereren voor passagiers  

---

-	Hoe ik de projext heb op gebouwd:
De code is opgesplitst in verschillende packages:

- A. Main
  Bevat het menu en de hoofdlus van de applicatie.

- B. Personen  
  Bevat de klassen voor passagiers en personeelsleden.

- C. Trein
  Bevat de klassen voor locomotieven, wagons en treinen.

- D. Reis  
  Bevat de logica voor reizen en ticketverkoop.

- E. Ticket
  Bevat informatie over het ticket, de passagier en de klasse.

---

Over de  Boardinglijst  -->
Voor elke passagier met een ticket kan er een boardinglijst worden aangemaakt.  
Deze boardinglijst wordt opgeslagen als een tekstbestand en bevat de belangrijkste gegevens van de reis en de passagier.
